
# Team Blog App

App intended to be used on smartphone.
![Project preview](./src/preview/preview.jpg)


&nbsp;
 
## 💡 What has been used
- React library
- TypeScript
- React Router DOM
- React Hook Form
- MUI
- Firebase

&nbsp;

## 💿 Installation

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```

&nbsp;
